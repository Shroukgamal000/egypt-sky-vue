**Aura** is a high-fidelity meteorological intelligence platform designed to provide precise, real-time weather data across Egypt. Developed with a focus on modern aesthetics and data accuracy, Aura offers a premium experience for monitoring atmospheric conditions.

Developed and maintained by **Shrouk Gamal Kamel**.

## ‚ú® Features

- **Real-time Meteorological Intelligence**: Live data fetching for temperature, humidity, wind speed, and more.
- **5-Day Precise Forecast**: High-resolution forecasting for the upcoming week.
- **Hourly Atmospheric Charts**: Dynamic visualization of temperature patterns using Recharts.
- **Advanced Health Metrics**: Real-time tracking of Air Quality Index (AQI), UV Index, and visibility.
- **Bilingual Interface**: Seamless search functionality supporting both English and Arabic city names.
- **Premium Glassmorphism Design**: A state-of-the-art UI utilizing backdrop blurs, subtle gradients, and high-contrast typography.
- **Responsive Architecture**: Fully optimized for desktop, tablet, and mobile workstations.

## üõ†Ô∏è Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [Bun](https://bun.sh/) or [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd egypt-sky-vue
   ```

2. **Install dependencies**
   ```bash
   bun install
   # or
   npm install
   ```

3. **Start the development server**
   ```bash
   bun run dev
   # or
   npm run dev
   ```

4. **Build for production**
   ```bash
   bun run build
   # or
   npm run build
   ```
